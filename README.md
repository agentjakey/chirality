# Chirality Atlas: Star Ascidian Edition

UCSD Vibe Coding Active Matter 2026 Hackathon

**Can two local rules generate a living star pattern?**

---

## What this is

A toy geometric model of *Botryllus schlosseri* (star ascidian) colony spatial organization.

The model reproduces two things: the quasi-regular spacing of star centers across a substrate, and the radial arm geometry within each star. It does this using only local physical rules, no global templates.

**What this is not.** This is not a full Botryllus developmental model. It does not include biochemistry, blastogenic staging, colonial immune recognition, 3D substrate attachment, or emergent arm count. Arm number (n_arms=7) is a model parameter. See Scope and Limitations below.

---

## Key results

- Radial order >= 0.8 at default parameters (agents well-confined at target radius)
- Swirl score rises from ~0.01 at omega=0 to ~0.3 at omega=2.5 (chirality is measurable)
- Arm structure is preserved under chirality up to omega ~2 before blurring
- Six distinct morphological phases identified across the parameter space
- Center spacing is regular without any explicit center-repulsion rule

The main finding: a Turing activator-inhibitor field places star centers. Active agents with angular repulsion form discrete arms. Chirality adds measurable swirl without immediately destroying the arm geometry.

## Scope and limitations

This is a toy geometric model. It models the visible spatial signature of a Botryllus colony, not the full organism.

**What the model reproduces:**
- Quasi-regular star center spacing (from Turing instability)
- Radial arm confinement at r_target (radial_order >= 0.8)
- Discrete arm structure (from angular repulsion between arm groups)
- Measurable chirality (swirl_score)

**What it does not reproduce:**
- Arm count is a parameter (n_arms=7), not an emergent quantity
- No Botryllus biochemistry or signaling molecules
- No blastogenic developmental cycle or staging
- No colonial immune recognition (self/non-self fusion)
- 2D only -- no substrate curvature or 3D geometry
- Parameter values are not fit to organism measurements

**Known metric limitation:** The arm_count metric uses angular histogram peak detection. At the default density of 3 agents per arm, it reads 1-3 even when 7 visual arms are present. Use n_per_arm >= 5 for reliable arm count detection, or rely on radial_order as the primary quality metric.

## Model overview

### Layer 1 -- Gierer-Meinhardt field

Turing instability in two chemical species. Short-range activation, long-range inhibition.

```
da/dt = Da*lap(a) + rho*a^2 / (h*(1 + kappa*a^2)) - mu_a*a + rho_0
dh/dt = Dh*lap(h) + rho*a^2 - mu_h*h
```

Turing condition: `Dh >> Da`. Inhibitor diffuses much faster than activator, causing the uniform state to break into quasi-periodic spots. Those spots become star center positions. No explicit center-repulsion rule is needed -- spacing emerges from the diffusion ratio.

Numerical scheme: IMEX (implicit diffusion in Fourier space, explicit reaction). Unconditionally stable for the diffusion term. Default `Dh/Da = 100`.

### Layer 2 -- Active zooid agents

Self-propelled particles with five force contributions:

```
F_attract  = -k_a * r_vec                       (toward center)
F_radial   = k_r * (r_target - r) * r_hat       (spring to r_target)
F_angular  = -k_ang * (1 - |dphi|/sigma) * sign(dphi) * t_hat  (arm repulsion)
F_ev       = excluded volume
dtheta/dt  = omega + sqrt(2*Dr) * noise         (chirality + rotational diffusion)
```

The angular repulsion is the key ingredient. For pairs in the same star but different arm groups, it pushes groups apart until they reach equal angular spacing. Without it: a ring. With it: discrete arms.

## LLM use

Two contributions that changed the design:

1. **Two-layer decomposition.** We described the biological pattern and asked what minimal mechanisms explain the two-level spatial organization. The LLM proposed separating center placement (a colony-level Turing field) from arm formation (local active agents). This was not the initial design. The single-layer alternative required ad hoc center-repulsion rules that the Turing field provides automatically.

2. **Anti-cheat metrics.** Early simulations looked like stars visually but failed on inspection. The LLM proposed a hierarchy of five metrics, each targeting a specific failure mode: radial_order (catches rings without arms), arm_count (catches featureless rings), swirl_score (cannot be faked by a non-rotating pattern), fragmentation (catches escaped agents). Each was verified against known failure cases.

Both contributions were checked, tested, and verified before use. All equations, parameter values, biological scope, and limitations were human decisions.
