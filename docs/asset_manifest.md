## Weakest Assets to Inspect Before Presenting

- **`simulation_vs_target_features.png`**: arm count metric reads 1.5 arms (not 7) because find_peaks needs more agents per arm or stronger angular repulsion. The radial_order=1.0 tells the real story. Be ready to explain the metric limitation.
- **`slide3_simulation_sequence.png`**: if the simulation only found 2 centers (K=2), the "sequence" shows modest arm separation. Mention this is the minimal-compute version; n_field_steps=3000 at N=64 gives cleaner spots.
- **`phase_transition_parameter_sweep.gif`**: 2 fps is slow; speed it up in the browser or increase fps in `04_make_movies.py`.
- **`center_spacing_inhibition_ratio.png`**: spacing_quality values are low if the GM field at N=32 doesn't fully equilibrate in 1500 steps. Mention the full-res version uses N=64, n_steps=3000.
