# Final Slide Export Notes

## Primary deliverable

File: `outputs/slides/Chirality_Atlas_Star_Ascidian_Edition.pptx`
Generated by: `python scripts/make_pptx.py`
Size: ~440 KB

This PPTX contains 5 slides with embedded panel images.
All images are the pre-generated panel PNGs from `outputs/panels/`.

## To regenerate

```
python scripts/make_pptx.py
```

Requires python-pptx: `pip install python-pptx --break-system-packages`

## To export to PDF

Option 1 (recommended on Windows): Open the PPTX in PowerPoint, File > Export > Create PDF/XPS

Option 2 (LibreOffice if installed):
```
soffice --headless --convert-to pdf outputs/slides/Chirality_Atlas_Star_Ascidian_Edition.pptx
```

Option 3 (if presenting from macOS):
```
osascript -e 'tell application "Keynote" to ...'  # not applicable, wrong format
# Use: open the PPTX in Keynote or Preview after import
```
