## Project Summary

Chirality Atlas: Star Ascidian Edition models the spatial geometry of Botryllus schlosseri
colonies using a two-layer hybrid model. Layer 1 is a Gierer-Meinhardt Turing field
that places star centers. Layer 2 is an active zooid agent model that forms radial arms
around those centers. Chirality (omega) is the primary control knob: it measurably rotates
arm geometry without destroying arm structure.

   Cause: 3 agents per arm is below reliable peak detection threshold.
   Response: present radial_order (> 0.8) as the primary metric; mention arm count
   limitation honestly. The geometry is correct even if the metric is noisy.

2. **Phase diagram at reduced resolution.**
   Cause: N=32 and short runs for speed.
   Response: acknowledge in slide 4 that boundaries are indicative.

3. **GIF file sizes may be large.**
   If Pillow is available, GIFs are typically 1-5 MB. Reduce fps or frame count
   in 04_make_movies.py if needed.

4. **slide3_simulation_sequence.png may show only 2-3 centers.**
   Cause: fast preset uses smaller grid; GM may find fewer spots.
   Response: mention that the full-resolution run finds 4-6 centers.
